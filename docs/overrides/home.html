Features

25+ Built-in Modules

File I/O, HTTP, SSH, JSON, YAML, templates, concurrency, retry, and more — batteries included.

Kubernetes Native

First-class k8s module in the Cloud edition: deploy, scale, rollout, and apply manifests with zero YAML.

Python-like Syntax

Starlark is a Python dialect — familiar, deterministic, and safe. No learning curve for Python developers.

Built-in Test Runner

kite test with parallel execution, assertions, filtering, and setup/teardown hooks.

Safe by Default

Sandbox mode restricts dangerous operations. Trust mode allows everything. Fine-grained allow/deny rules.

Embeddable Runtime

Embed starbase in any Go app to add scripting. 27 modules included, permission sandboxing, custom modules, optional WASM plugins.

Quick Install

go install github.com/project-starkite/starkite@latest

Download from GitHub Releases

git clone https://github.com/project-starkite/starkite.git
cd starkite
go build -o kite .
